# JustyBase for VS Code

This repository is the public information and support hub for the JustyBase VS Code extensions.

The source code is maintained in a private development repository. This public repository is used for documentation, issue tracking, release notes, and Marketplace support links.

## Marketplace

- Core extension: search for "JustyBase Core" in the VS Code extensions view

Install from Visual Studio Marketplace (search for "JustyBase Core").

## Support

Use this repository issues to report bugs, request features, or ask support questions. Please do not include database passwords, connection strings, private SQL, customer data, or proprietary schema details in public issues.

## Privacy

JustyBase runs inside VS Code and connects to databases configured by the user. AI-assisted features may send selected SQL, schema context, diagnostics, or result previews to GitHub Copilot/Microsoft services only when those features are used. Review `PRIVACY.md` before enabling AI workflows in corporate environments.

## Repository Scope

This public repository intentionally does not contain the extension source code. It exists so Marketplace links, issue tracking, release notes, and project ownership point to the JustyBase organization rather than a personal source repository.
